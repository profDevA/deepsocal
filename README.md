# DeepSoCal

A regional strategic design agency building transformative experiences for Southern California's brands by blending deep research, AI & technology, strategy, design, and technology.

## Live Site

[deepsocal.com](https://deepsocal.com)

## Tech Stack

- HTML5
- CSS3 / Bootstrap 5
- JavaScript / jQuery
- Owl Carousel
- Font Awesome 6

## Project Structure

```
├── index.html              # Homepage
├── home-2.html             # Homepage (alternate layout)
├── about-us.html           # About page
├── works.html              # Portfolio listing
├── works/                  # Case study detail pages
│   └── forever-a-surfer.html
├── services/               # Service landing pages
│   └── strategy-inetgration.html
├── css/                    # Stylesheets
├── js/                     # Scripts
├── img/                    # Images, videos, SVGs
└── fonts/                  # Custom web fonts (Druk, ES Allianz)
```

## Local Development

No build step required. Open `index.html` in a browser, or run a local server:

```bash
python -m http.server 8080
```

Then visit `http://localhost:8080`.
